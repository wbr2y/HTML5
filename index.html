Definições e outros conteúdos

Título # Descrição
HTML5 -#- Linguagem de marcação para definição de propriedades da estrutura de páginas web.
Metadatas -#- Ficam no elemento <head>, contém as informações da página, configuração para o SEO, e inclui os estilos (CSS) e escritos (JS).
Sessão-root -#- É o elemento <body>, o qual representa o conteúdo do documento HTML exibido para o usuário.
Elementos -#- São as "tags" utilizadas para fazer a identificação da estrutura e semantica da página para o navegador.
